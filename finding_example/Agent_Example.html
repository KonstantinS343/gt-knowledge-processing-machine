Задачей агента поиска примеров для заданного понятия является нахождение примеров заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска примеров для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти примеры.
